# Tech Stack

## Frontend
- **HTML5:** Modern semantic structure.
- **Vanilla JavaScript (ES6+):** Core logic, DOM manipulation, and real-time calculations.
- **Tailwind CSS (via CDN):** High-speed utility-first styling and layout.

## Typography & Assets
- **Plus Jakarta Sans:** Primary sans-serif font for readability and modern branding.
- **JetBrains Mono:** Technical monospace font for precise time and data representation.
- **Lucide/SVG Icons:** Lightweight, scalable vector graphics for UI actions.

## Storage & Performance
- **LocalStorage API:** Client-side persistence for selected cities and custom hours.
- **RAF/Interval Sync:** Real-time synchronization of the current time marker.
